# Plazma Browser

## Plans for Plazma Browser
========================

Features
--------

*   Smooth HTML rendering
*   CSS styling
*   Usable for everyday use
*   Javascript execution
*   Tab support
*   Bookmarks
*   History
*   Settings
*   Customisation
*   Less then 500mb of memory
